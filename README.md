# BookStore API

REST API для управления книгами на Go.

## Стек

- Go 1.25
- Gorilla Mux
- GORM v2
- MySQL 8.0
- Docker

## Быстрый старт

```bash
git clone https://github.com/Tim73916/bookstore-management.git
cd bookstore-management
docker-compose up -d

API доступно по адресу: http://localhost:9010
